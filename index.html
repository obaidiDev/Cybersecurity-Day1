Foundation 01

The CIA triad

Three promises every system makes — and constantly trades off.

What we protect

Confidentiality · Integrity · Availability

🔒

Confidentiality

Only the right people can see it. Broken by leaks, snooping, stolen data.

✅

Integrity

The data is correct and untampered. Broken by fraud, corruption, silent edits.

⚡

Availability

It’s there when you need it. Broken by outages, DDoS, ransomware.

Security isn’t maxing all three — it’s balancing them for the situation. Let’s feel that tension.

Don’t define and move on — the next slide lets them manipulate the trade-off directly.

Foundation 02

Who’s attacking — and why

Motive shapes method. Read the motive and you can predict the move.

The line-up

Five kinds of adversary

🧑‍💻

Script kiddie

Motive: ego, thrill. Off-the-shelf tools, noisy, low skill.

📣

Hacktivist

Motive: ideology. Wants a message seen — defacement, leaks.

💰

Cybercriminal

Motive: money. Ransomware, fraud, data for sale.

🎫

Insider

Motive: revenge, greed. Already has legitimate access.

🛰️

Nation-state / APT

Motive: espionage. Patient, stealthy, well-funded.

    Foundation 05 · the board

    MITRE ATT&CK

    The shared vocabulary that lets every analyst on Earth describe an attack the exact same way.

    Why a shared vocabulary

    “They ran a script” vs. T1059

    Without it

    Everyone describes the same move differently. Reports don’t line up, tools don’t agree, knowledge doesn’t transfer.

    With ATT&CK

    Every technique has an ID and a definition. T1566 means “phishing” to everyone — SOCs, vendors, threat reports.

    Tactics = the attacker’s goal (the why).  Techniques = how they achieve it (the how), each with an ID like T1566.

    Keep it concrete. Tactic = column = goal. Technique = the ID inside it. The board on the next slide is explorable + it’s our weekly tagging surface.
    The board · explore, then tag every attack

    Hover a technique for its meaning · click to tag it for the selected day · Day 1 is pre-tagged from the attack we just walked. Coverage builds all week.

    Day 1 already carries the retail intrusion’s techniques — the same ones from the map. Days 2–5, switch the day and tag each new attack. The dots show the week’s coverage growing.
